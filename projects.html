A-Fetch

A-Fetch is a utility for unix based systems to display their system configuration in a minimal manner. I took inspiration from computer parts & circuitry to create this logo.
More information can be found on the project
Project Github





Bryte

Bryte is a utility to control display backlighting for unix based systems.
I wanted to keep this logo simple and I achieved that by integrating the Unicode symbol for brightness into the application name.
More information can be found on the project
Project Github
